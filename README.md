Simulated Trading Platform

Virtual stock trading environment featuring FTSE350 listed companies from the London Stock Exchange.

<img width="450" height="250" alt="image" src="https://github.com/Roland-02/sTrade/assets/111765814/381ce00d-eaed-4038-9ae2-a84f8fad6792">
<br></br>

Users can buy/sell stock shares with virtual currency. Market prices are manipulated manually by algorithm, or using up-to-date current share prices, allowing for trading outside of market hours.

<img width="250" height="275" alt="image" src="https://github.com/Roland-02/sTrade/assets/111765814/5f9392cc-c643-46d4-b6be-14073527c9fd">
<br></br>

The application features a Banking system so user's can acquire more in-game currency for trading - loan payments are taken from the user's balance and can default if there is a lack of funds

<img width="450" height="250" alt="image" src="https://github.com/Roland-02/sTrade/assets/111765814/ad03011e-970b-4841-9476-99c5fbca1ca9">
<br></br>

An LTM score also tracks and shows the user's progress/success in trading.
